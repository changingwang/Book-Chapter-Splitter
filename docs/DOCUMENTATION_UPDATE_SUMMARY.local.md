## 新文件名格式

### 章节文件
- **格式**: `第X章_标题.md`
- **示例**: `第一章_马克思主义政治学的对象和方法.md`

### 小节文件
- **格式**: `X.Y_标题.md`
- **说明**: X=章节编号，Y=小节编号
- **示例**: 
  - `1.1_马克思主义政治学的形成.md`
  - `1.2_马克思主义政治学的对象.md`
  - `2.1_人是社会存在物.md`

## 格式优势

1. **有序排列**: 文件按逻辑顺序自然排列
2. **清晰层次**: 明确显示章节和小节的层次关系
3. **易于导航**: 便于用户快速定位和浏览
4. **标准化**: 统一的命名规范，便于维护
